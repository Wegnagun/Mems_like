#### Windows

`python -m venv venv`  
`source venv/bin/activate`  
`python -m pip install --upgrade pip `  
`pip install --upgrade setuptools ` опционально...  
`python -m pip install --upgrade pip setuptools` либо так)
`pip install -r requirements.txt`
